# 🛒 ReBazar - Backend

## 🌐 Project Overview

ReBazar Backend is a RESTful API built with **Node.js**, **Express.js**, and **MongoDB**. It powers the marketplace by handling authentication, products, orders, payments, dashboards, and role-based authorization for buyers, sellers, and administrators.
